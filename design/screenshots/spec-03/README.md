## 📋 截圖步驟

### 準備工作
1. 啟動 Android 模擬器
2. 運行應用：`flutter run -d emulator-5554`
3. 確保網絡連接正常

### 截取流程

#### 1. 未下載狀態
```
1. 在書籍列表中選擇一本未下載的書
2. 進入書籍詳情頁
3. 等待頁面完全加載
4. 截圖保存為 01-not-downloaded.png
```

#### 2. 下載中狀態
```
1. 在上一個頁面點擊「開始下載」
2. 等待進度條顯示（約 10-50%）
3. 截圖保存為 02-downloading.png
```

#### 3. 已下載狀態
```
1. 等待下載完成（或選擇已下載的書籍）
2. 等待「下載完成」Snackbar 出現
3. 截圖保存為 03-downloaded.png
```

#### 4. 網絡錯誤
```
1. 打開模擬器設置
2. 關閉網絡連接（WiFi 和移動數據）
3. 返回應用，進入未下載書籍詳情頁
4. 點擊「開始下載」
5. 等待錯誤提示出現
6. 截圖保存為 04-error-no-network.png
```

#### 5. 刪除確認
```
1. 在已下載書籍詳情頁
2. 點擊「刪除」按鈕
3. 確認對話框出現
4. 截圖保存為 05-delete-confirmation.png
```
