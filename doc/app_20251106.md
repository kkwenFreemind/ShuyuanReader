# 製作ShuyanReader APP

- 採用Flutter打造Android平台的電子書閱讀器

## 儲存位置

- catalog/books.json 存放在https://github.com/kkwenFreemind/ShuyuanReader/blob/main/catalog/books.json

- covers/*.png 存放在https://github.com/kkwenFreemind/ShuyuanReader/blob/cover/

- epub3/*.epub 存放在https://github.com/kkwenFreemind/ShuyuanReader/blob/epub3/


## APP

### Use Case1

    - App 啟動時透過 http.get() 下載 books.json
    - 用 ListView 或 GridView 顯示書籍封面 + 書名 + 作者

### Use Case2

    - 點選書籍 → 下載 EPUB 並存入本地
    - 儲存到 App 沙盒（getApplicationDocumentsDirectory()）
    - 儲存後立即進入閱讀頁

### 整合 EPUB 閱讀器

    - 原生解析 + 自製閱讀 UI
